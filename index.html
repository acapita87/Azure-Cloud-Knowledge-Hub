☁️ Infrastructure as a Service (IaaS)

Compute

Virtual Machines, App Service, Container Instances e Kubernetes (AKS) para deploy de aplicações.

  • Azure Virtual Machines (VMs)
  • Virtual Machine Scale Sets (VMSS)
  • Azure Container Instances (ACI)
  • Azure Kubernetes Service (AKS)

Storage

Soluções de armazenamento escalável para dados, backups e disaster recovery.

  • Blob Storage (Object Storage)
  • File Shares (SMB/NFS)
  • Managed Disks
  • Archive Storage

Networking

Conectividade, routing e segurança de rede para infraestruturas distribuídas.

  • Virtual Networks (VNets)
  • Load Balancers
  • Application Gateway
  • ExpressRoute & VPN

Database

Bases de dados geridas para OLTP e OLAP com escalabilidade automática.

  • Azure SQL Database
  • MySQL/PostgreSQL
  • Cosmos DB (NoSQL)
  • Data Lake Storage Gen2

Backup & Disaster Recovery

Proteção de dados e continuidade de negócio com Azure Backup e Site Recovery.

  • Azure Backup
  • Azure Site Recovery (ASR)
  • Geo-redundant Storage
  • Data Box para migração

Management & Monitoring

Ferramentas para gerir, monitorar e otimizar infraestruturas Azure.

  • Azure Monitor
  • Log Analytics
  • Azure Automation
  • Policy & Blueprints

  • SharePoint Online

  • Dynamics 365 Finance & Operations
  • Project Operations

Azure App Service

  • Web Apps (ASP.NET, Node.js, Python)
  • Mobile Apps
  • API Apps
  • Logic Apps (Workflow Automation)

Functions & Serverless

Compute serverless para processamento de eventos e funções orientadas por gatilhos.

  • Azure Functions
  • Event Grid
  • Service Bus
  • Queue Storage

Analytics & Big Data

Plataformas para análise, BI e processamento de grandes volumes de dados.

  • Synapse Analytics
  • Power BI
  • Data Factory (ETL)
  • Databricks

AI & Machine Learning

Serviços de IA e ML para modelos preditivos e cognitive services.

  • Azure Machine Learning
  • Azure AI Services (OpenAI, Vision, Language)
  • Cognitive Search
  • Bot Service

🔐 Azure Security & Identity

Identity & Access (Zero Trust)

Autenticação forte, MFA, PIM e Conditional Access para acesso seguro.

  • Azure Entra ID (AAD)
  • Multi-Factor Authentication (MFA)
  • Privileged Identity Management (PIM)
  • Conditional Access
  • Identity Protection

Network Security

Segurança de perímetro com firewalls, NSGs e DDoS protection.

  • Network Security Groups (NSGs)
  • Azure Firewall
  • DDoS Protection
  • WAF (Web Application Firewall)
  • Private Endpoints

Data Protection

Criptografia, TDE, BYOK e proteção de dados em repouso e em trânsito.

  • Azure Key Vault
  • Transparent Data Encryption (TDE)
  • Bring Your Own Key (BYOK)
  • SQL Always Encrypted
  • Data Classification

Compliance & Governance

Ferramentas para auditoria, compliance e governança de recursos.

  • Azure Policy
  • Azure Blueprints
  • Compliance Manager
  • Audit Logs (Azure Activity Log)
  • Security & Compliance Center

Threat Detection & Response

Detecção de ameaças, SIEM e resposta a incidentes de segurança.

  • Microsoft Defender for Cloud
  • Microsoft Sentinel (SIEM)
  • Threat Intelligence
  • Security Alert Management
  • Incident Response

Application Security

Segurança ao nível da aplicação com DevSecOps e vulnerability scanning.

  • Defender for DevOps
  • Code Scanning & Analysis
  • Container Security
  • API Security
  • SAST & DAST Tools

  3. Privileged Access Management (PIM)
  4. Network Security - NSGs, Firewalls, DDoS Protection

  6. Compliance frameworks - GDPR, ISO 27001, SOC 2
  7. Microsoft Defender for Cloud - assessment e hardening
  8. Microsoft Sentinel - SIEM, detecção de ameaças e hunting
  9. Incident Response e Disaster Recovery

💡 Dicas para Certificação

  • Estude oficial: Microsoft Learn (learn.microsoft.com) - cursos gratuitos
  • Practice Exams: Faça testes práticos no Microsoft Learning Portal
  • Hands-on Labs: Pratique no Azure Free Account (200€ créditos)
  • Study Groups: Junte-se a comunidades e grupos de estudo
  • Real-world Experience: Trabalhe com Azure em projetos reais
